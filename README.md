# Erlenmeyer

Flask boilerplate

## Usage

1. Get `cookiecutter`
  ```bash
  $ pip install cookiecutter
  # You should be using pipsi though ;)
  $ pipsi install cookiecutter
  ```

2. Initialize your app
  ```bash
  $ cookiecutter git@github.com:SiliconValleyInsight/erlenmeyer.git
  ```

3. Up
  ```bash
  $ cd <app_name>
  $ ./up.sh
  ```
